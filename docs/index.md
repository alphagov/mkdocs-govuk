# MkDocs GOV.UK Theme

This is a theme for using MkDocs with GOV.UK style

## Requirements

* python v3.8 and above 
* MkDocs

## Usage

* Copy the govuk_theme to your project root
* Set the [theme] configuration option in your `mkdocs.yml` config file.

```yaml
theme:
    name: null
    custom_dir: 'govuk_theme'
```
